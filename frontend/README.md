# MusicUI_Search

### How to use
```
# if already install nodejs and npm
sudo apt remove nodejs npm
sudo curl -fsSL https://deb.nodesource.com/setup_15.x | sudo -E bash -
sudo apt-get install -y nodejs
sudo apt-get install build-essential
```

load js script lib
```
npm install
```

start local site
```
npm start
```
